基础cpu介绍

cpu

一颗cpu好坏现在主要看：

主频
外频
缓存
指令集
是否支持睿频

inter cpu 从低到高分为：

  1. Atom 凌动 for 上网本、平板
  2. Celeron 赛扬
  3. Pentium 奔腾
  4. Core 酷睿
  5. Xeno 至强 for 服务器

现在常用的酷睿系列又分为：

  1. i3 双核4线程，集成gpu，无睿频基数
  2. i5 四核4线程，有睿频技术
  3. i7 四核8线程，高性能，高功耗

cpu的线程 和 核心数
核心分为：

  1. 每核心单线程
  2. 每核心双线程

线程是指cpu可以同时处理的进程数量

每个cpu型号尾部的字母：

  1. y 超低功耗版，不支持睿频，适用上网本
  2. u 低电压版，适用超极本
  3. m 标准电压版
  4. h 同m一样，焊在主板上，不可更换
  5. mq 移动标压四核版，一般是酷睿i7

同型号cpu又分：

  1. 笔记本版
  2. 台式版
