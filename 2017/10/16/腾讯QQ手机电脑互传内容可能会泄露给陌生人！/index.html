安全隐患：腾讯QQ手机电脑互传内容可能会泄露给陌生人！

今天用手机QQ收到一群友转发的一篇好文章，看完后觉着有必要收藏PC端。

然后果断点击分享 -> 搜索好友 -> 我的电脑，然后弹出了好多个【我的电脑】？？！

认真看还是有区别的，有【功能】、【联系人】之分。

但暂时网页标签不支持分享到【我的电脑】，所以只有【转发】功能：

这种情况下点击转发，再搜索【我的电脑】显示的列表只能是联系人中的【我的电脑】。

情景还原：
看到一片好文章 -> 分享 ->　搜索【我的电脑】-> 选中第一个 -> 确定

过了十几秒，【我的电脑】回了一句：“啥玩意儿？”

【我的电脑】竟然自己说话了？！！ 握！！grass！！

虽说这对老手来说很容易避免，但长辈们使用多数不熟练，完全有隐私信息转发陌生人的可能，已提交qq建议/反馈。
